# v1: "내일의 질문, 오늘 답하기"

## 🎯 핵심 컨셉

**"내일의 질문, 오늘 답하기"**

시간을 역전시킵니다.
면접 질문을 미리 알고, 오늘부터 준비합니다.

---

## 💡 컨셉의 의미

### 고객이 진짜 원하는 것
```
❌ "면접 질문 40개"  (드릴)
✅ "면접장에서 당당하게 답하기"  (구멍)
```

### 컨셉의 힘
```
애플 iPod: "주머니 속의 1000곡"
→ 공간의 역전 (1000곡이 주머니 속에)

QueryDaily: "내일의 질문, 오늘 답하기"
→ 시간의 역전 (내일 질문을 오늘 준비)
```

### 차별점
```
일반 면접 준비:
"면접 보고 → 뭐 물었는지 기억 → 다음엔 잘해야지"
→ 사후 학습 (이미 늦음)

QueryDaily:
"이력서 분석 → 질문 예측 → 미리 준비 → 면접 가기"
→ 사전 학습 (선제적)
```

---

## 📋 서비스 본질 재정의

### 우리가 하는 것
1. **이력서 분석** → 당신의 경험/기술 파악
2. **질문 예측** → 면접관이 물을 질문 생성
3. **매일 발송** → 오전 7시, 저녁 5시
4. **STAR 가이드** → 구조화된 답변 방법

### 고객이 얻는 것
```
Before: "뭐 물어볼지 몰라서 불안해요"
After: "이미 예상하고 답 준비했어요"

감정 변화:
불안 → 예측 가능 → 준비됨 → 자신감
```

## 🎯 핵심 상품 2개 (선택과 집중)

### 1. 크리티컬 히트 ₩9,900
**"내일의 핵심 3질문, 오늘 답하기"**

**제공 내용**:
- 이력서 기반 핵심 질문 3개
- 각 질문별 STAR 기법 답변 가이드
- 결제 후 24시간 내 이메일 발송

**타겟**:
- "일단 어떤 건지 경험해보고 싶어요"
- "면접 며칠 안 남았는데 긴급 대비"

**메시지**:
```
가장 중요한 3가지 질문을 예측합니다.
오늘 답을 준비하세요.
```

### 2. 그로스 플랜 ₩49,000 [MOST POPULAR]
**"내일의 모든 질문, 20일간 답하기"**

**제공 내용**:
- 20일간 매일 2개 질문 (총 40개)
- 모든 질문에 STAR 기법 답변 가이드
- 매일 오전 7시, 저녁 5시 자동 발송
- 꼬리 질문 대비 팁

**타겟**:
- "제대로 준비하고 싶어요"
- "모든 질문에 대비하고 싶어요"

**메시지**:
```
이력서의 모든 경험을 질문으로 예측합니다.
20일간 매일 답을 준비하세요.
```

**차별점 명확화**:
```
ChatGPT vs QueryDaily

ChatGPT:
- 일반적 질문 ("Spring이 뭔가요?")
- 내 이력서와 무관
- 매번 프롬프트 작성

QueryDaily:
- 이력서 기반 ("3번 프로젝트에서 Spring을 왜?")
- 내 경험 맞춤
- 이력서 한 번만 올리면 매일 자동
```
