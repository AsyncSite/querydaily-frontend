# QueryDaily 제품 상세 가이드

## 브랜드 철학

### "성장의 에너지를 낮춰서, 모두가 자신의 경험을 무기로 만들 수 있게 한다."

면접 준비가 어려운 이유는 실력이 부족해서가 아닙니다.
단지, 너무 많은 에너지를 요구하기 때문입니다.

이력서를 펼치고, 질문을 찾아보고, 답변을 정리하는...
이 모든 '시작'이 너무 무겁습니다.

QueryDaily는 이 **'시작의 에너지'**를 제거합니다.

매일 아침 이미 하고 있는 **'이메일 확인'**에
면접 준비가 자연스럽게 녹아듭니다.

커피 마시며, 출근하며, 자연스럽게.

## 철학의 구조

브랜드 철학은 3단계로 작동합니다:

### 1. 철학 (Philosophy)
```
"에너지를 낮춰 접근성을 높인다"
```

### 2. 제도 (System)
```
자동화된 루틴
- 매일 아침 9시 자동 배송
- 이메일로 편하게 수신
- 루틴에 자연스럽게 통합
```

### 3. 물건 (Product)
```
이메일로 배송되는 맞춤형 면접 질문
```

## 에너지 레벨 가이드

### ☕ (Very Low)
- 출근하며 이메일 확인
- 시간: 1-5분
- 예: 일일 복습, 간단한 확인

### ☕☕ (Low)
- 커피 마시며 읽고 생각
- 시간: 10-30분
- 예: 그로스 플랜, 크리티컬 히트, 레주메 핏

### ☕☕☕ (Medium)
- 집중해서 읽고 정리
- 시간: 30-60분
- 예: 라스트 체크

### ☕☕☕☕ (High)
- 실전처럼 집중 필요
- 시간: 90분
- 예: 리얼 인터뷰 (유일)

## 공통 특징

모든 제품은 다음 원칙을 따릅니다:

### 1. 이력서 기반 맞춤화
```
당신의 경험에서 나올 수 있는 질문만 제공
검색하면 나오는 일반 질문 X
```

### 2. 자동화된 루틴
```
매일 아침 9시 자동 배송 (그로스 플랜)
이메일로 편하게 수신
루틴에 자연스럽게 통합
```

### 3. 에너지 최소화
```
시작의 에너지를 제거
이메일 확인만으로 충분
출근하며 생각하면 됨
```

### 4. 실전 중심
```
모범 답변보다 생각하는 법
암기보다 이해
외우기보다 정리
```
