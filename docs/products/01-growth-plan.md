- **가격**: ₩49,000 (정가 ₩106,000)
- **기간**: 20일 집중 훈련
- **배지**: MOST POPULAR

## 철학

### "20일이면, 면접이 기대되기 시작합니다."

우리는 면접 준비가 '한 번에 몰아서'가 아니라
'매일 조금씩 쌓아가는 것'이라고 믿습니다.

하지만 매일 준비하는 게 어려운 이유는
시작할 때마다 많은 에너지가 들기 때문입니다.

이력서를 펼치고,
오늘 뭘 준비할지 고민하고,
자료를 찾아보는...

이 모든 시작이 너무 무겁습니다.

그로스 플랜은 이 '시작의 에너지'를 제거합니다.

매일 아침 9시, 이메일 하나가 도착합니다.
오늘 생각할 질문이 이미 준비되어 있습니다.

당신은 그냥 읽고, 생각하면 됩니다.
커피 마시며, 출근하며, 자연스럽게.

20일 후, 당신은 깨닫게 됩니다.
'나도 매일 준비할 수 있구나.'
'내 경험도 충분히 무기가 되는구나.'

이것이 그로스 플랜이 존재하는 이유입니다.

## 작동 방식

### Day 0 (지금, 5분만 투자)

```
📝 이력서 업로드
   → AI가 당신의 경험 분석
   → 20일 맞춤 질문 생성
   → 끝. 이후는 모두 자동
```

---

### Week 1 (기본기 다지기)

#### ☕ Day 1 (월요일 아침)

```
07:45 📧 이메일 도착
      "Redis를 도입한 이유가 무엇이었나요?"

08:00 출근하며 생각 (에너지: ☕)
      "응답 속도 개선... DB 부하 줄이려고..."

22:00 📝 간단히 메모 (5분)
      + 📚 모범 답변 가이드 확인
```

#### ☕ Day 2 (화요일 아침)

```
07:45 📧 이메일 도착

```

